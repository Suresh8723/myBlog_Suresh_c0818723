1. Lets Volunteer

A Social platform where the Volunteer who is willing to contribute to society can view and contribute to the volunteering events organized by Organization. There are not many platforms that connect volunteers and organizers and they do not cater to all types of volunteering. Our application is the one-stop solution for bringing together volunteers and organizers that may serve any cause across different locations.

2. Notes Application

Notes Application is design to take the daily notes under certain categories and user has to create a category and a note can be created using that category, user can also see the multiple notes that are taking under single category. User has to add title, description and also have options to add location, images and audio to every single note. User can see the different categories on the landing page and when the user clicks on one of the category and user will be able to see the notes that are created under that category.
